# CV Builder (React)

A simple CV / Resume builder built with **React**.

This application allows users to enter their personal details, education, and work experience, preview the CV in real time, and download it as a PDF.

## ðŸ–¨ Downloading the CV

To download the CV as a PDF:

1. Click the **Download CV** button in the preview section
2. In the print dialog, select **Save as PDF**
3. Disable **Headers and Footers** in print settings for a clean CV
